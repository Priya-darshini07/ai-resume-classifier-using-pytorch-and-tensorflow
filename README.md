# AI Resume Classifier (PyTorch & TensorFlow)

This project demonstrates an end-to-end text classification pipeline
implemented using both PyTorch and TensorFlow.

## Features
- Resume text classification into job domains
- Bag-of-Words text vectorization
- PyTorch implementation with manual training loop
- TensorFlow (Keras) implementation with high-level API
- Google Colab compatible

## Tech Stack
- Python
- PyTorch
- TensorFlow (Keras)
- Scikit-learn
